# Hotel Booking Analysis

## 📊 Overview
This project analyzes hotel booking data to uncover booking trends, customer behavior, and cancellation patterns.

## 🛠 Tools Used
- Power BI
- Python
- Excel

## 🔍 Key Objectives
- Analyze seasonal booking trends
- Study cancellation rates
- Evaluate customer segments

## 📈 Key Insights
- Higher cancellations in certain booking channels
- Seasonal peaks affect occupancy rates
- Lead time impacts cancellation probability
